Mobile

Recommended

Install directly from your browser — no app store needed. Works on iPhone, iPad, and Android. Plays offline once installed.

  1. 1 Open ryanscarbel.com/kitten-crawler-man in Safari
  2. 2 Tap the Share button (box with arrow) at the bottom of the screen
  3. 3 Scroll down and tap "Add to Home Screen"
  4. 4 Tap Add — the game icon appears on your home screen
  5. 5 Open it once while online. After that, it works offline.
  1. 1 Open ryanscarbel.com/kitten-crawler-man in Chrome
  2. 2 Tap the ⋮ menu (top-right) and choose "Add to Home screen" or "Install app"
  3. 3 Tap Install
  4. 4 Open it once while online. After that, it works offline.
How it works: Your browser downloads the entire game (~30 MB of images and audio) in the background on first visit. After that, everything runs from your device — no internet required.

Desktop

Download the full game as a ZIP file and run it locally on your Mac, Windows PC, or Linux machine.

⬇ Download ZIP (~30 MB)
  1. 1 Download and unzip the file
  2. 2 Mac: double-click start.command
    Windows: double-click start.bat
    Linux: run python3 -m http.server 8080
  3. 3 The game opens automatically in your browser at localhost:8080
Why a server? The game's audio engine uses browser fetch(), which is blocked on local files for security reasons. The launcher script starts a tiny local server that takes care of this. Python 3 is pre-installed on Mac; Windows users can get it at python.org.
Or play in the browser →
Chrome / Edge on desktop? You can also install the game as a desktop app — look for the install icon (⊕) in the address bar after visiting the game, or use the browser menu → "Install Kitten Crawler Man". It works offline just like the mobile version.

Common questions

Will my progress be saved?
Progress is saved in your browser's local storage. It persists between sessions as long as you don't clear your browser data. The installed app and the browser version share separate save files.
Does the offline version get updates?
The installed PWA checks for updates automatically in the background whenever you have internet. The ZIP download is a fixed snapshot — re-download it to get updates.
The Mac launcher is blocked by macOS security
Right-click (or Control-click) start.command and choose Open, then click Open again in the dialog. You only need to do this once.
The game doesn't load audio on Windows
Make sure you're launching with start.bat (not by opening index.html directly). If Python isn't installed, download it from python.org and run the installer.
